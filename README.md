# Ethereum Simple Storage Smart Contract

This project contains a basic Ethereum smart contract written in Solidity and deployed using Remix IDE.

## 📌 Overview
The `SimpleStorage` contract demonstrates core blockchain concepts including:
- Storing and retrieving data on-chain
- Smart contract deployment
- Function interaction (read & write)
- Contract ownership using `msg.sender`

## ⚙️ Technologies Used
- Solidity
- Ethereum Virtual Machine (EVM)
- Remix IDE

## 🚀 Features
- Set and retrieve a number
- Update a message string
- Track contract owner

## 🧪 Testing
The contract was compiled and deployed in Remix, and tested using:
- `getNumber()` to retrieve stored values
- `setNumber()` to update values
- `setMessage()` to modify stored text
